УСТАНОВКА адаптивного топ-бара для ЛК

1) Залейте файлы в проект:
   - /includes/topbar.php
   - /assets/js/topbar.js

2) Подключите на страницах личного кабинета сразу после <?php include 'includes/header.php'; ?> (или вместо вашей текущей шапки):

   <?php
     $user_login = $user['login'] ?? null; // если у вас переменная $user доступна
     // Кастомизируйте пункты меню при желании
     $menu = [
       ['href'=>'/deals/create.php','label'=>'Создать сделку','primary'=>true],
       ['href'=>'/dashboard/profile.php','label'=>'Профиль'],
       ['href'=>'/dashboard/balance.php','label'=>'Вывести'],
       ['href'=>'/notifications/index.php','label'=>'Уведомления'],
     ];
     include __DIR__.'/../includes/topbar.php';
   ?>

3) Внизу шаблона (перед </body>) подключите скрипт:
   <script src="/assets/js/topbar.js"></script>

Что это даёт:
- На мобилках: кнопки не «плывут», а становятся горизонтальной прокруткой «пилюль» + бургер с выдвижной панелью.
- На десктопе: остаётся привычный ряд кнопок.
- Для iPhone добавлена безопасная «safe area» и липкий верх.

При необходимости кнопки счётчиков (Активные/Успешные/Отклонённые/Арбитраж) ниже оставьте без изменений — они уже адаптивные.
